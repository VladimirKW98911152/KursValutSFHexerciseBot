# KursValutSFHexerciseBot
# В данном репозитории содержатся три файла Python, в которых содержится код Телеграммбота KursValutSFHexerciseBot, который был составлен в ходе выполнения итогового заданияя 6.1(PJ-02) юнита 6/9 Итоговый проект модуля C 12. Итоговый проект по ООП "Телеграм-бот". В файле main.py содержатся обработчики сообщений. В файле configuration.py содержатся токен телеграмбота а так же словарь, в котором в качестве ключей используются названия валют а в качестве значений используются их тикеры. В файле extensions.py содержится класс get_price - который принимает три аргумента и возвращает нужную сумму в валюте. При написании использовались встроенная библиотека json, а так же установленные библиотеки requests и pytelegrambotapi. Для получения курса валют использовался API сервиса cryptocompare (https://min-api.cryptocompare.com/data/price?fsym=EUR&tsyms=RUB). Следуя условию задания, я использовал следующие переменные: base, которой присваивалось имя валюты, цену на которую надо узнать; quote, которой присваивалось имя валюты, цену в которой надо узнать; amount, которой присваиволсь значение количества переводимой валюты.
